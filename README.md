 🎬 Movie Recommendation System

 📌 Description
This project recommends movies based on user input using Machine Learning.

 ⚙️ Setup Instructions

1. Download required file:
similarity.pkl → https://drive.google.com/file/d/11A_c8CjRvV73T1mzN89FLFQHMX7bRTU5/view?usp=sharing

2. Place the file in project folder

3. Install dependencies:
pip install -r requirements.txt

4. Run the project:
python app.py

 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
